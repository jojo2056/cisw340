Week 1

Project 1 — Create the index.html & upload to the Student Web Server

This project will illustrate your ability to code XHTML/CSS and Upload to the Student Web Server.

These are expected competencies when you begin this class, as you should have knowledge to satisfy the prerequisite requirements.

Before you begin this project, you should view the presentation in D2L for Week 1.

PROJECT 1 REQUIREMENTS:

  1. Code an index.html page which provides:
    1. DOCTYPE DECLARATION STRICT
    2. Box model using a banner, left nav and right content area (or something using at least 3 divisions)
    3. Comments that identify sections in the HTML file and CSS file
    4. An introduction to your website with your First Name and Last initial ONLY. Do not include your last name or student ID.
    5. Create an image banner for the page (use a banner maker like cooltext.com or some other free provider).
    6. Your banner image should not be more than 60kb.
    7. Create a heading to briefly state your educational goals (use h1)
    8. Create a heading for Class Assignments (use h1)
  2. All assignments will be uploaded to the server, and the index.html page should link to each assignment. Use an unordered list to provide links to the following and format using CSS:
    1. Week 1 — Project 1 (links to index.html)
    2. Week 2 — Project 2
    3. Week 3 — Project 3
    4. Week 4 — Project 4
    5. Week 5 — Project 5
    6. Week 6 — Project 6
    7. Week 7 — Project 7
    8. Week 8 — Final Project

DIRECTORY STRUCTURE REQUIREMENTS:

Remember that all files must be uploaded “into” the public_html folder.
All files and folders must be lowercase with no spaces

  1. FOLDER public_html
  2. HOMEPAGE inside public_html index.html



Create subfolders for each project. Organize files associated with that week's assignment in the subfolder for the week. Your directory will look like this:

  1. public_html (folder at the root of your WebServer space)
  2. index.html (file)
  3. week1 (folder)
  4. week2 (folder)
