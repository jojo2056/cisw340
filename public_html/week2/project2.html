Project 2: Pseudo-Classes and Pseudo Elements

This project includes two pseudo elements: :first-letter and :first-line. The first line is lilac the first letter is a large italic red-purple. The second line is normal.


CSS code:
p.element:first-letter {font-family: cursive; font-size:300%; color:#8A2BE2;}
p.element:first-line {color: #6c7bff; font-size: 200%;}


HTML code:
<p class="element">List of State Flowers and Flower Gallery<br /> Click on the state flower name to view the photo below.</p>
