Project 2 — Using Pseudo-classes and Pseudo Elements - Create a new page

Pseudo-classes: link, visited, hover and active

NOTE: a:hover MUST come after a:link and a:visited
NOTE: a:active MUST come after a:hover

a:link {color:#FF0000;}   /* unvisited link - green */
a:visited {color:#00FF00;}   /* visited link - red */
a:hover {color:#FF00FF;}   /* mouse over link - pink */
a:active {color:#0000FF;}   /* selected link - blue*/
