Week 2

Project 2 — Using Pseudo-classes and Pseudo Elements - Create a new page

PROJECT 2 REQUIREMENTS:

  1. Code a new page, called project2.html, to include the following:
    1. DOCTYPE DECLARATION STRICT
    2. Utilize the Box model, using at least 3 divisions
    3. Comments that identify sections in the HTML file and CSS file




  2. To practice using the items of the discussion this week, create content of your choice to incorporate the following:
    1. Pseudo-classes: link, visited, hover and active
    2. Pseudo-classes: use at least 2 structural pseudo classes
    3. Use one target pseudo-class
    4. Use at least one pseudo elemen





  3. The content of your page can be anything, but these techniques could be well utilized if you had a story to tell. Write a page about any topic that interests you, and use Pseudo-classes and Pseudo-elements to do some formatting.

  4. Upload your project to the server, and assure the project links from the index.html page.
