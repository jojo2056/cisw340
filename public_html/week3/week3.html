Week 3

Project 3 — Using @font-face

PROJECT 3 REQUIREMENTS:

  1. Code a new page, called project3.html, to include the following:
    1. DOCTYPE DECLARATION STRICT
    2. Utilize the Box model, using at least 3 divisions
    3. Comments that identify sections in the HTML file and CSS file




  2. Create a webpage utilizing Web typography methods introduced, and also using the CSS3 methods for using fonts.
    1. Specify at least 2 different fonts with @font-face
    2. Specify at least 3 variations on the display of the font
    3. Make first letters of a column or a paragraph capitalized, like you would see in a newspaper or magazine article.
    4. Use at least 1 image to complement the page, and assure you use alt, height and width for the image.







  3. The content of your page can be anything, but these techniques could be well utilized in a tutorial, or an informational story of some kind. You could write it like a blog if you wanted. Pick a topic that interests you and have fun with the assignment.

  4. Upload your project to the server, and assure the project links from the index.html page.

  5. Include the required directory structure as outlined on page 2

  6. If you have questions, post them in the Instructor Office
