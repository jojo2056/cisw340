Week 5

Project 5 — Using CSS Columns

PROJECT 5 REQUIREMENTS:

  1. Code a new page, called project5.html, to include the following:
    1. DOCTYPE DECLARATION STRICT
    2. Utilize the Box model
    3. Comments that identify sections in the HTML file and CSS file




  2. Create a webpage utilizing Web typography methods introduced, and also using the CSS3 methods for using columns. Include browser prefixes to support the browsers that current support this technique.
  3. Column requirements include:
    1. Minimum 3 column layout
    2. Use of header for the columns
    3. Define specific width
    4. Alignment of text - justify
    5. Use at least 1 image to complement the page, and assure you use alt, height and width for the image.







  4. The content of your page can be anything, but these techniques could be well utilized in a tutorial, or an informational story of some kind. You could write it like a blog if you wanted. Pick a topic that interests you and have fun with the assignment.

  5. Upload your project to the server, and assure the project links from the index.html page.

  6. Include the required directory structure as outlined on page 2

  7. If you have questions, post them in the Instructor Office
